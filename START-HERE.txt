🎯 ARCHITECTURE OVERVIEW
─────────────────────────────────────────────────────────────────────────────

  Your Terminal
      ↓
  .\launch-claude-code.ps1 -Local
      ├─ Start proxy on localhost:8000
      ├─ Generate/retrieve API key
      └─ Launch 'claude code' with env vars:
         • ANTHROPIC_BASE_URL=http://localhost:8000
         • ANTHROPIC_API_KEY=sk_xxxxx
         • ANTHROPIC_MODEL=claude-sonnet-4-6
      ↓
  Claude Code CLI
      ↓
  HTTP POST localhost:8000/v1/messages
      ├─ Proxy translates Anthropic → OpenAI format
      ├─ Validates API key
      ├─ Routes model via MODEL_MAP
      └─ Rate limiting, logging, CORS
      ↓
  Ollama (localhost:11434)
      ├─ Loads model into VRAM (first request: ~30s)
      ├─ Runs inference on your GPU
      └─ Streams response back
      ↓
  ← Response bubbles back to Claude Code CLI
      ↓
  You see normal Claude Code output

🔧 CONFIGURATION
─────────────────────────────────────────────────────────────────────────────

Default .env (created by setup script):

  KEYS_FILE=keys.json
  AUTH_ENABLED=true
  RATE_LIMIT_ENABLED=true
  CORS_ENABLED=true
  OLLAMA_BASE_URL=http://localhost:11434
  MODEL_MAP=claude-sonnet-4-6:qwen3-coder:30b,\
            claude-opus-4-6:deepseek-r1:32b,\
            *:qwen3-coder:30b


Custom Model Mapping (edit .env):

  # Route all Claude models to one specific model:
  MODEL_MAP=*:qwen3-coder:30b

  # Route different Claude models to different Ollama models:
  MODEL_MAP=claude-sonnet-4-6:qwen3-coder:30b,\
            claude-opus-4-6:deepseek-r1:32b,\
            claude-haiku-4-5:qwen3-coder:8b

  After editing, restart Claude Code to pick up changes.


🚨 QUICK TROUBLESHOOTING
─────────────────────────────────────────────────────────────────────────────

Problem: "Claude Code CLI not installed"
  Fix: npm install -g @anthropic-ai/claude-code

Problem: "Port 8000 in use"
  Windows: netstat -ano | findstr :8000
           taskkill /PID <PID> /F
  Linux/macOS: lsof -i :8000; kill -9 <PID>

Problem: "Proxy failed to start"
  Check logs: cat logs/proxy.log or Get-Content logs/proxy.log -Wait

Problem: "Model not found"
  Ensure Ollama is running: ollama run qwen3-coder:30b

Problem: "First request slow (5-15s)"
  Normal — model is loading into VRAM. Subsequent requests are fast.

Problem: "API key authentication failed"
  Generate new key: python scripts/generate_api_key.py --email you@example.com --department dev

For more help: See CLAUDE-CODE-QUICKSTART.md#troubleshooting
