HOW IT WORKS
================================================================================

The launcher checks for:
  ✓ .env configuration file
  ✓ keys.json API key storage
  ✓ Python dependencies (fastapi, uvicorn, etc.)

If anything is missing:
  → Automatically runs setup
  → Shows progress
  → Continues to launch Claude Code

If everything is ready:
  → Skips setup
  → Goes straight to launch


QUICKEST START: JUST RUN THIS
================================================================================

Windows (PowerShell, CMD, or Git Bash):
  python run-claude-code.py

That's literally it! The launcher will:
  1. Detect Windows OS
  2. Check what setup is needed
  3. Auto-run any missing setup
  4. Start proxy
  5. Launch Claude Code

ERROR HANDLING
================================================================================

If setup fails:
  → Clear error message shows what went wrong
  → Suggests how to fix it
  → Does NOT proceed to launch Claude Code

Examples:
  - "Python dependencies failed to install"
    → Check: pip install -r requirements.txt
  - "Setup script not found"
    → Check: setup-claude-code.ps1 exists
  - "PowerShell not found"
    → Windows comes with PowerShell, but try: choco install powershell
